## Description

NestJS Boilerplate TypeORM

## Installation

```bash
# Install package
$ npm install

# Install husky
$ npx husky install
```

## Running the API

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash

# unit tests
$ npm run test:unit

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
